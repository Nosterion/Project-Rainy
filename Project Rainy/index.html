Introduction

This is a site for a sort-of pet project of mine, a visual novel called 'Rainy Days'. Hence the name of the site: 'Project Rainy'. Visual novels written by westerners are usually made for windows with the help of a script language, usually Python. Or more often using an engine that is based on some script language, for example Ren'Py. (which is also Python based) Visual novels can also be browser-based. The Monogatari visual novel engine is a good example of that, using Javascript as a programming language. If you don't know any programming language, TyranoBuilder Visual Novel Studio allows you to make awesome stuff on multiple platforms. I prefer to micro-manage everything myself, and create a common platform for the upcoming game and community feedback also, so my go-to choice is Monogatari and Javascript at the moment.

I like coming up with new ideas and creating stuff. But my talents are limited. I have my moments in storywriting, and know a few programming languages to an extent. (Python, Javascript, Java) I also fancy photography. But I have no talent for drawing whatsoever. Kind of a handicap for a wannabe visual novel creator who likes doing all the stuff himself. At one point, I will need graphic artists and sound artists, but first I would like to establish the things I can do on my own. For example making background images for visual novels out of real life pictures. If it works, it would also define the base style, as a kind of signiture style. But there is no point if no one likes it. So, esteemed visitor, please express your opinion in the Gallery section by selecting the style you prefer. Visual novels are not made for solely entertaining its creator, after all. ☺

Gallery

There are two variety of every picture. One is made with colored pencil and the other is with oil paint. I made the pictures with a Canon SX260 HS camera, edited them with Corel PaintShop Pro 2018 and transformed them with Corel Painter Essentials 6 using a variety of additional layers for optimal result. I learned the techique as I worked on the pictures, so some of them are better than the others as I gained more experience working with the tools.

At default, the pictures are shown in their original form. By clicking on them, you will get the option to see the 2 transformed styles and make your vote. After that, your selected style will become the new default one for that picture.
